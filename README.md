# Telegram Shop Bot: Furniture from China 🇨🇳

Масштабируемый Telegram-бот для продажи PDF-каталогов мебели из Китая с интеграцией ЮKassa и Битрикс24.

## 🚀 Основной функционал
- **Витрина каталогов:** Иерархическое меню (Категории -> Подкатегории -> Файлы).
- **Диплинки (Deep Linking):** Персонализированные ссылки для Reels (индивидуальное приветствие + выбор категории).
- **Автоматизация оплат:** Интеграция ЮKassa (нативные платежи Telegram).
- **CRM Интеграция:** Автоматическая передача лидов в Битрикс24 через API.
- **Админ-панель:** Управление ценами, текстами бота, контентом и ссылками без изменения кода.
- **Гибкость:** Поддержка отправки MediaGroup (несколько PDF-файлов в одном сообщении).

## 🛠 Технологический стек
- **Python 3.11+**
- **Aiogram 3.x** (Асинхронный фреймворк)
- **SQLAlchemy 2.0** (ORM для работы с БД)
- **SQLite** (Легкая база данных)
- **Aiohttp** (Веб-сервер и запросы к API)

## ⚙️ Настройка
1. Скопируй `.env.example` в `.env` и заполни переменные:
   ```env
   BOT_TOKEN=твой_токен
   PAYMENT_TOKEN=токен_юкасса
   ADMIN_IDS=твой_id
   BITRIX_WEBHOOK_URL=ссылка_вебхука

Установи зависимости:

pip install -r requirements.txt

Запусти бота:

python main.py

🏗 Структура проекта

tg_bot/ — ядро проекта
database/ — модели данных и миграции
handlers/ — обработчики сообщений (админка, магазин, формы)
keyboards/ — генераторы Inline и Reply меню
services/ — интеграция с внешними API
